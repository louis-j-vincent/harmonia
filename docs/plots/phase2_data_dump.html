Analysis for Phase 2

Observation 1: Systematic Degradation (Hypothesis A support)

A1 (87.5%) → A2 (50%) → B (37.5%) → C (20%) shows a clear degradation pattern. This could reflect inference model's loss of accuracy in harder sections (solos, rhythmic complexity).

Observation 2: B's Offset Conflict (Hypothesis B support)

B at offset 0 (musically correct per user's ear) has only 12.5% match, while offset -3 has 37.5%. Yet offset -3 places B too early (17.5s vs expected 21.5s). This could indicate GT alignment is off, not that B's true position scores poorly.

Observation 3: Confidence Paradox (Hypothesis B support)

B's inferred chords are highly confident (0.796–0.850 avg) despite having poor root-match. A good inference system should show low confidence when wrong. High confidence + low match suggests either: (a) model is overconfident, or (b) GT is misaligned and model is "right."

Observation 4: C's Ambiguity

C has 20% match across offsets -1, 0, +1, showing no clear preference. This could mean: (a) C is fundamentally hard to infer, or (b) C's GT position is uncertain.
